# Roles And Flows

## Usuario

```text
Registro
-> Perfil
-> Publicar solicitud
-> Recibir postulaciones
-> Comparar profesionales
-> Seleccionar
-> Coordinar
-> Presupuesto
-> Trabajo
-> Confirmacion
-> Calificacion
```

## Profesional

```text
Registro
-> Perfil profesional
-> Rubros
-> Radio de trabajo
-> Disponibilidad
-> Ver oportunidades
-> Postularse
-> Ser seleccionado
-> Coordinar
-> Presupuestar
-> Ejecutar
-> Finalizar
-> Recibir calificacion
```

## Administrador

```text
Validar profesionales
-> Moderar usuarios
-> Gestionar categorias
-> Revisar denuncias
-> Resolver reclamos
-> Suspender cuentas
-> Auditar acciones
```
